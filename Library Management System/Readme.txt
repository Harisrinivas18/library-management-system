U have to import the database into maria db in the Xampp first then you can start implementing the project
Follow these steps to deploy the system
- Open XAMPP server and Press start on Apache and MYSQL
- Open the Php my admin page from the xampp in any web browser.
- Import the database into the phpMyAdmin (under databases press import and select the database named lms from the local computer)
- Paste the project folder into your local drive xampp htdocs folder.
- The sample address of the folder is:
- C:\xampp\htdocs\Library Management System
- You can run the library management with the help of editors or else you can also use
the direct link is as follows: http://localhost/Library%20Management%20System
- You can use the functionalities of library management system with the help of above
steps
